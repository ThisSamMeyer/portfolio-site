What I Do

In software engineering, “the stack” is all of the different programs and applications that are working together to respond to requests from the user and form a fully functional website or application. A full-stack developer works with the "front-end" user interface as well as the "back-end" server and database communications. Full-stack development is creating from the ground up and connecting all the pieces along the way.
